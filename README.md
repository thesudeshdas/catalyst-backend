## Run Locally

1 - Clone the project

```bash
  git clone https://github.com/thesudeshdas/catalyst-backend.git
```

2 - Go to the project directory

```bash
  cd catalyst-backend
```

3 - Install dependencies

```bash
  npm install
```

4 - Add environment variables

```bash
  PORT
  DB_URL
```

5 - Start the server

```bash
  npm run start
```
